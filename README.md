# 什么是区块链？

区块链是由*不可变的*、*有顺序记录的*区块组成。  
区块可以包含交易数据、文件数据或者其它你想要记录的数据。不过最重要的是这些区块通过哈希表链接在一起。

# 什么是散列？

散列函数是一个输入值函数，从该输入创建一个确定输入值的输出值。
